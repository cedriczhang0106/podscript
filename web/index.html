[ 07 — FAQ ]

常见问题

要花钱吗？
转录默认在你自己电脑上跑，离线、免费、不上传。只有当你想要更聪明的 AI 摘要时，才需要接一个大模型的 key——DeepSeek V4、Kimi K2.6、智谱 GLM-5.2 都很便宜，一集播客的摘要成本通常就几分钱。不接也照样能用，只是少一段摘要。
一定要用 Obsidian 吗？
不必。输出的是普通 Markdown，落到任意文件夹都行——只是对 Obsidian 用户特别顺手，转完直接在库里可搜可链。
会上传我的音频吗？
不会。音频全程在你机器上转录，转完即删，原始稿留档可重洗。就算你接了大模型做摘要，发出去的也只是转好的文字、不是音频——而且发往的是你自己填的那个模型服务。
落点会乱放吗？
不会。首次运行先问你"落到哪个文件夹"，记住后复用、可随时改——工具不替你猜，别人的库结构各异。
转录一集要多久？
主要花在本地 Whisper 听写上，一小时的播客约 10–20 分钟，取决于你的电脑；接了大模型再加几分钟做清洗、摘要、分段。同一条链接重转会复用已转录稿、跳过听写，几十秒就好。
接大模型怎么接？推荐哪个？
点「接大模型」，选一个预设（DeepSeek / Kimi / 智谱 GLM），把你自己的 API key 粘进去保存即可——地址和模型名会自动填好。要又快又稳，Kimi 选 moonshot-v1-128k。key 只存在你本地浏览器、不经第三方。
逐字稿能按话题分段吗？
能。接了大模型时，会按话题把全文切成若干段、每段配一个小标题，并在段内自动分成小段落；左侧还会生成可点击的大纲，点标题跳转、滚动时悬浮跟随。纯本地（没接模型）则按每 10 分钟分段。
转录报错、或接了模型没生效？
逐字稿顶部会显示一条红色提示，写明真实原因（如 key 不对、模型名不对、没接通）。最常见是接模型时忘了粘 key——预设只自动填地址和模型，key 要你自己粘。照提示改了重转即可。

在线模型 / ONLINE LLM

接一个模型，把摘要做得更好

转录始终在你本地免费跑——这一步只把转好的文字交给大模型做断句、去口水和摘要。国产模型便宜、中文好，填好下面三项即可。

Key 只存在你自己的浏览器本地、随请求发往你填的那个服务，不经过任何第三方。留空保存 = 退回纯本地（只转录、不做 AI 摘要）。

落点 / OUTPUT FOLDER

转录稿落到哪个文件夹？

填你的 Obsidian vault 路径，转录稿会直接写进去。留空 = 存到服务器的 ./output。这个设置只存在你自己的浏览器本地。

以后想改：直接编辑页面上「落点 / OUTPUT」那一行即可。
